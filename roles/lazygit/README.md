# Lazygit

Installs [Lazygit](https://github.com/jesseduffield/lazygit), a terminal UI for git.

- **macOS**: Homebrew
- **Ubuntu**: GitHub releases
- **Fedora**: GitHub releases (via `github_release` role)

Integrates with Neovim via `<leader>gg` keybinding.

## Usage

```bash
dotfiles -t lazygit
lazygit  # Run in a git repo
```
